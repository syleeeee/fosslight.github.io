FOSSLight

When you develop and distribute software using open source software,
you can use FOSSLight for governing open source.
FOSSLight Scanner can perform the open source analysis of source code, binary and dependency and generate the report that contains the open source information that can be extracted.
FOSSLight Hub helps you use the open source software compliantly and securely by managing open source, license and also vulnerability. Also it can process the OSC (Open Source Compliance) process by managing BOM for each project and track issues on open source license obligations and vulnerabilities.
